# SamQ's Dotfiles

My Arch setup as of December 2025 uses the following:

- Window manager: [MangoWC](https://mangowc.vercel.app/), [Hyprland](https://hypr.land/)
- Launcher: [Rofi](https://github.com/davatorium/rofi)
- Bar: [Waybar](https://github.com/Alexays/Waybar)
- Wallpaper daemon: [swaybg](https://github.com/swaywm/swaybg)
- Idle daemon: [swayidle](https://github.com/swaywm/swayidle)
- Lock screen: [hyprlock](https://github.com/hyprwm/hyprlock/)
- Notifications: [Mako](https://github.com/emersion/mako)
- Screenshots: [Flameshot](https://flameshot.org/)
- Authentication Agent: lxsession (GTK3)
- Audio: PipeWire + [wiremix](https://github.com/tsowell/wiremix)
- Shell: [Zsh](https://zsh.org) + [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting) + [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)
- System monitor: [Btop](https://github.com/aristocratos/btop), [fastfetch](https://github.com/fastfetch-cli/fastfetch)
- Monospace font: [Jetbrains mono nerd](https://www.nerdfonts.com)
- GTK theme: [Materia dark](https://github.com/nana-4/materia-theme)
- Terminal: [Foot](https://codeberg.org/dnkl/foot)
- Editor: [Neovim](https://neovim.io/)
- File manager: pcmanfm (GTK3), [Yazi](https://github.com/sxyazi/yazi)
- Primary web browser: [ungoogled-chromium](https://github.com/ungoogled-software/ungoogled-chromium)
- Image viewer: [swayimg](https://github.com/artemsen/swayimg)
- Media player: [mpv](https://mpv.io/)
