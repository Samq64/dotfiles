# Neovim Config

This configuration uses Neovim's built-in package manager to manage plugins. The following packages should be installed globally in addition to the LSP servers and formatters listed in [lsp.lua](lua/lsp.lua).

- Neovim 0.12+
- A clipboard tool (`wl-clipboard` on Wayland)
- A [Nerd Font](https://www.nerdfonts.com/) and a terminal that can render it
- tree-sitter-cli
- fzf
- ripgrep (optional)
- fd (optional)
